#### Publish package

1. Npm login:

    ```bash
    npm login --scope=@nplus.tech --registry=https://registry.npmjs.org/
    ```
   
2. Update version to second in `package.json`:
   ```
   Example: "version": "1.0.2" => "version": "1.0.3"
   ```

3. Publish package:

    ```bash
    npm publish
    ```
